About Me:

I am research scholar in Computer Science and Engineering department at IIT Hyderabad under the supervision of Dr. Antony Franklin A. I am working on Mobile Edge Computing(recently changed to Multi-access Edge Computing). MEC can be seen as a cloud server running at the edge of a mobile network and performing specific tasks that might not be possible with traditional network infrastructure, i.e., providing the real time network information. Mobile-edge Computing is characterized by on-premises, proximity to the users, lower latency, location awareness, and network context information. These characteristics enable MEC to be used for deployment of various applications and services, namely edge content delivery, data aggregation, content optimization, edge analytics, offloading, etc. Mobile Edge Computing servers can be deployed LTE macro base station (eNodeB), 3G Radio Network Controller (RNC) site, and Multi-Radio Access Technology (RAT) cell aggregation site.
